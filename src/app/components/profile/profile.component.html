About GitHub Portfolios

GitHub Portfolios is a dynamic website that allows you to search for any GitHub developer and displays publicly shared information about them. This website uses the official GitHub API to scrape information such as basic profile information, uploaded and starred repositories and renders it in the style of an online portfolio for that user. The purpose of this web application is to search for developers easily and see their contributions on GitHub.

The GitHub API has a rate limit of 5000 requests per hour. If your search returns an error, the API limit might have been reached for your device. For more information about API rate limiting, see REST API GitHub documents.
